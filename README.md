# Simple text comparison program
## About
* Use `--help` for help.
* The `-m` argument indicates the operation option.
* Example:
`diff "Their color a deep purple hue" "Their color is a deep shade of purple"`
* Or you can compare the program output to the console with the expected value
`diff "D:\Download\test.toml" "D:\Download\a.exe" -m program`
